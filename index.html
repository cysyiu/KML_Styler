KML Styler

This tool is for bridging the Google Earth KML with other GIS software.
Import a KML file to visualize it on the map. If the KML contains inline styling (stored in elements such as <IconStyle>, <LineStyle> or <PolyStyle>), the control panel is automatically populated with those values. Click "Simplify Styles" to remove the sepeartion of <styleID>-normal and <styleID>-highlight. You may adjust the style parameters below to dynamically change the feature styles and styleID. When finished, click "Export KML" to output a KML file that contains the updated styles.
